# Portfolio Listing

## Project title
TaskFlow Pro — Team Productivity SaaS

## One-line description
A production-minded PHP and MySQL task management platform with teams, Kanban, analytics, calendars, reports, files, notifications and REST API access.

## Technologies
PHP 8, MySQL/MariaDB, PDO, JavaScript, HTML5, CSS3, Chart.js, Docker, REST API, PWA.

## Resume bullets
- Built a full-stack SaaS-style task management application with secure authentication, role-based authorization and relational MySQL architecture.
- Implemented drag-and-drop Kanban, calendar planning, analytics dashboards, file uploads, comments, checklists and team collaboration.
- Developed REST API CRUD endpoints, CSV/PDF-friendly reporting, audit logs, notification polling and Docker-based deployment.
- Applied CSRF protection, password hashing, prepared statements, ownership validation and secure file MIME checks.
