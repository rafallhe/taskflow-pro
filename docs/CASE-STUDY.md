# TaskFlow Pro — Portfolio Case Study

## Problem

Individuals and small teams often manage tasks across disconnected notes, spreadsheets and chat messages. This creates poor visibility, missed deadlines and inconsistent follow-up.

## Solution

TaskFlow Pro is a full-stack productivity platform built with PHP and MySQL. It centralizes task planning, progress tracking, reporting, calendars, attachments, comments, checklists, user roles and API access.

## Key Features

- Secure authentication and role-based access
- Dashboard with interactive analytics
- Task CRUD with priorities, categories, progress and tags
- Drag-and-drop Kanban workflow
- Monthly deadline calendar
- Reports with CSV export
- Task comments, checklists and attachments
- Notifications and overdue alerts
- User profile and password management
- Login audit history
- REST-style API token access
- Responsive dark/light interface

## Technical Highlights

- PDO prepared statements
- Password hashing
- CSRF protection
- Ownership checks
- Foreign keys and indexes
- Secure upload validation
- Automatic base URL detection
- Responsive CSS architecture

## Portfolio Description

TaskFlow Pro demonstrates full-stack PHP development, relational database design, secure authentication, role authorization, dashboard analytics, responsive interface design and production-minded application structure.
